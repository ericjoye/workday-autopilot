# Workday Autopilot — Chrome Web Store Listing

## Extension Name

**Workday Autopilot — Auto-Fill Job Applications**

## Short Description (132 chars max)

Auto-fill and auto-submit Workday job applications with one click. Save your profile once, apply to hundreds of jobs in seconds.

## Long Description

Workday Autopilot is a Chrome extension that eliminates the tedium of filling out Workday job applications. Set up your profile once — name, email, phone, address, LinkedIn, work experience — and let Autopilot handle the rest. It fills every field and submits the application in about 3 seconds.

**HOW IT WORKS**
1. Install the extension and click the rocket icon to open your profile
2. Fill in your information (9 fields + work experience)
3. Visit any Workday-powered career page
4. Click the Autopilot button (or press `Ctrl+Shift+A`)
5. Watch as every field is filled and the application submits — automatically

**SMART FIELD DETECTION**
Autopilot doesn't just match field names — it uses multi-strategy detection across labels, placeholders, ARIA attributes, autocomplete values, and input types. It handles text inputs, email, phone, dropdowns (like state selectors), and work experience entries. React-compatible value injection ensures fields are actually filled, not just visually changed. This is what sets Autopilot apart from Chrome's built-in autofill, which breaks on Workday's complex JavaScript forms.

**TIERED PRICING**
- **Free**: 5 submissions per month — no credit card required
- **Pro** ($14.99/mo): Unlimited submissions, priority support
- **Team** ($29/mo): Shared profiles for recruiters and career coaches

**PRIVACY FIRST**
Your profile data stays in your browser (`chrome.storage.local`). Nothing is sent to external servers. The extension only activates on Workday career pages (`*.workday.com`, `*.myworkdayjobs.com`).

**KEYBOARD SHORTCUT**
Press `Ctrl+Shift+A` on any Workday page to trigger Autopilot instantly.

**COMPATIBILITY**
Works on all Workday-powered career pages including Workday Community, Workday Recruiting, and custom Workday career sites at Fortune 500 companies, universities, and government organizations.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `workday-autopilot/` folder
5. The Autopilot icon appears in your toolbar

## Requirements

- Chrome 88+ (Manifest V3 support)
